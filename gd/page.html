Daft Punk

Poster set

A passionate two-part tribute

to Daft Punk, this poster set

tries to emulate the texture,

patina, and style of vintage

band memorabilia. Featuring

each members iconic helmet

with bold typography and rich

analog textures, this project

is a homage to the official

posters of the band.

Measuring Tapes

Poster

A Poster made to practice

the use of textures and

explore the "Hopecore" style
